2 scala语言

  • 变量声明必须指明var\val var:可变，val:不可变

  • 声明时可以不指定数据类型，系统会自动判断，但需要注意是否符合要求，因为自动判断一般只有三种：（1）字面无小数点：Int；（2）字母有小数点的数字：Double；（3）字符串。

  • 如果指定数据类型：变量名后用":"，然后写数据类型；或者用数字后的类型标识。e.g. val i =6f

基本数据类型

String类型是通过默认导入java.lang包实现的，其他的是scala的相应包

算数运算符

与其他语言类同，与java相同。

关系运算符

与其他语言类同，与java相同。

逻辑运算符

与其他语言类同，与java相同。

位运算符

运算结果是数字，运算过程是对两个数字的对于二进制位进行逻辑操作，得到结果的对应二进制位的值。

位运算符有3种：与&、或|、异或^

赋值运算符

左移位并赋值；右移位并赋值；按位与赋值；按位异或赋值；按位或赋值。

返回值

语句块的最后执行的那一个语句的值，就是返回值。

语句块的划分：逻辑分隔，{}

val a={val c=2;val d=3;val c+d}

val a={val c=2;val d=3;val e=c+d}
a:Unit = () //声明和赋值同时 是没有返回值的

条件语句

两种写法：

(1)写在同一行

//java
int b;
if(a<0) b=0;
else b=100;

//scala
val b={if(a<0) 0 else 100}

//scala
val c={if(a<0) 0 else if(a<5) 10 else if(a<10 20 else 1000)}

(2)写在不同行

必须要使用集成开发环境

if在大多数情况下属于守卫用法：guard：在其他结构里嵌入if语句的应用方式。

现在已经支持if-else的写法，较早的版本不支持

循环语句

  • for的写法：
for(变量<-集合){循环体}

注意：变量不可以用val或var。

  • Range表达方式：起点 to 终点 或者 起点 until 终点 区别是是否包括终点：until 不包括。

没有step，步进就是1。

  • for中使用守卫(if)

写法：

for(变量<-集合 if(布尔表达式)){循环体}
  • 可以尽量减少嵌套层次

打印出1~10的偶数

object Main {
  def main(args: Array[String]): Unit = {
    val a=5;
    for(i<-1 to 10 if(i%2==0))
      println(i)
  }
}

打印出1~10的奇数，不要9 //守卫如果有多个条件，scala建议是直接写出，不写逻辑与

object Main {
  def main(args: Array[String]): Unit = {
    val a=5;
    for(i<-1 to 10 if(i%2==1) if(i!=9)) //scala常用的语法表示
      println(i)
  }
}

for的嵌套

循环默认没有break和continue；

如果一定要使用，需要手工导包；

for(最外层变量<-集合1；下一层变量<-集合2；…){循环体}

for(最外层变量<-集合1；下一层变量<-集合2 守卫…){循环体}

e.g.

for(i<-1 to 10 ;j<-11 to 13  if(i%2==0) if(i!=4) if(j!=13)) {循环体}
局限

e.g. 打印1~100内的素数，仍需要进行经典嵌套

object Main {
  def main(args: Array[String]): Unit = {
    val a=5;
    var flag = 1;
    for(i<-1 to 100 ) {
      flag=1
      for (j<-2 until i){
        if(i%j==0){
          flag=0;
        }
      }
      if (flag==1){
        print(i+" ")
      }
    }
  }
}

组合数据结构

Array

*较多使用

默认是定长数组。如果一定要使用非定长数组，需要自己导包。

写法与其他语言的不同之处：用圆括号。

访问使用索引。

Map

类似Python里面的字典，但：默认长度不可变（但python可变）。写法不同

写法：用圆括号，key与value之间用"->"

e.g. val a=Map("a"->1,"b"->2)

e.g. 为得到2

a(b)

Tuple

*较多使用

与其他语言相同。但访问方式比较特殊，用._序号形式访问。

特别注意：序号从1开始（不是索引[]，索引默认都是从0开始）。e.g.

val a=("a","b","c",1,2,3)

访问a中的第5个

a._5

数组中的数组

val b=(("a","b","c"),(1,2,3),7,8,9)

访问2中的第2.2个

b._2._2

索引访问

val c=Array("a","b","c",1,2,3)
c(4)
--res:Any=2

函数

普通函数

主要定义方法：

标准定义

def 函数名(参数列表)：返回值类型={函数体}

其中：参数列表写法：参数1:类型1,参数2:类型2,…;类型必须写，返回值类型可以不写。

一般不用return，因为最后执行的那一句的值就是返回值。

e.g.1 scala可以在main里定义一个新的函数

object Main {
  def main(args: Array[String]): Unit = {
    def add1(a1:Int,a2:Int):Int={a1+a2} //后一个Int可以不用写：如果系统可以正确自动判断；想自定义就需写明逻辑
    println(add1(1,2))
  }
}

匿名函数定义

作为函数的参数

以加法为例

val add2=(a1:Int,a2:Int)=>{a1+a2}

(参数列表)=>{函数体}

高阶函数

参数或者返回值是函数的函数。

def mul1(a1:Int,a2:Int):Int={a1*a2}
def highFunc(f1:(Int,Int)=>Int,a1:Int,a2:Int):Int={f1(a1,a2)}
print("*:");println(highFunc(mul1,1,2))

参数是函数

优点：函数功能更灵活、维护方便，编程任务分配简单。

基本写法：

def 函数名(形参列表):返回值类型={函数体}

形参列表中有函数形参，函数形参的写法：

函数名:函数类型;函数名:(形参类型列表)=>返回值类型

调用：函数形参，直接写函数名或者写匿名函数。

形参有默认值

没有默认值的形参要写在形参表的前面(必须传参)

def mul1(a1:Int,a2:Int):Int={a1*a2}
def add1(a1:Int,a2:Int):Int={a1+a2}
def highFunc(f1:(Int,Int)=>Int,a1:Int=1,a2:Int=2):Int={f1(a1,a2)}
print("*:");println(highFunc(mul1))
print("+:");println(highFunc(add1))

*函数形参默认

def mul1(a1:Int,a2:Int):Int={a1*a2}
def add1(a1:Int,a2:Int):Int={a1+a2}
def highFunc(f1:(Int,Int)=>Int=add1,a1:Int=1,a2:Int=2):Int={f1(a1,a2)}
print("*:");println(highFunc(mul1))
print("+:");println(highFunc())
调用时不按顺序传参

带形参名传参，明确指明要传哪个参数…

object Main {
  def main(args: Array[String]): Unit = {
    def mul1(a1:Int,a2:Int):Int={a1*a2}
    def add1(a1:Int,a2:Int):Int={a1+a2}
    def minus1(a1:Int,a2:Int):Int={a1-a2}
    def div1(a1:Int,a2:Int):Int={a1/a2}
    def highFunc(f1:(Int,Int)=>Int=add1,a1:Int=1,a2:Int=2):Int={f1(a1,a2)}
    print("+:  ");print(highFunc(a2=9))
  }
}
函数调用时传匿名函数

当作为参数的函数只用一次，并且写法比较简单的情况下，可以使用匿名函数；

object Main {
  def main(args: Array[String]): Unit = {
    def highFunc(f1:(Int,Int)=>Int,a1:Int=1,a2:Int=2):Int={f1(a1,a2)}
    print("*:  ");print(highFunc((a1:Int,a2:Int)=>{a1*a2}))
    print("+:  ");print(highFunc((a1:Int,a2:Int)=>{a1+a2}))
    print("-:  ");print(highFunc((a1:Int,a2:Int)=>{a1-a2}))
    print("/:  ");print(highFunc((a1:Int,a2:Int)=>{a1/a2}))
  }
}

练习：基础函数是def highFunc(f2:(Int,Int)=>Int,a1:Int=1,a2:Int=2)={f2(a1,a2)}，计算a*b/(a+b),a=5,b=8

object Main {
  def main(args: Array[String]): Unit = {
    def highFunc(f1:(Int,Int)=>Int,a1:Int=1,a2:Int=2):Int={f1(a1,a2)}
    print("*:  ");print(highFunc((x:Int,y:Int)=>{x*y/(x+y)}),5,8)
  }
}
函数调用时传使用_的匿名函数

匿名函数中_的使用前提：使用_不会影响参数判断，_所代表的参数在函数体中只使用一次。

例如函数：(a1:Int,a2:Int)=>{a1*a2}可以写成_*_ (下划线与声明时的形参按位置顺序对应)

object Main {
  def main(args: Array[String]): Unit = {
    def highFunc(f2:(Int,Int)=>Int,a1:Int=1,a2:Int=2):Int={f2(a1,a2)}
    print("*:  "); println(highFunc(_*_,2,3))
    print("+:  "); println(highFunc(_+_,2,3))
    print("-:  "); println(highFunc(_-_,2,3))
    print("/:  "); println(highFunc(_/_,2,3))
  }
}

返回值是函数

读文档时注意！

object Main {
  def main(args: Array[String]): Unit = {
    def highFunc(a1:Int=1):(Int,Int)=>Int={
      if(a1<0) (a1:Int,a2:Int)=>{a1*a2}
      else(a1:Int,a2:Int) =>{a1+a2}
    }
//    5*8 How to call highFunc
    print("*:  ");println(highFunc(-1)(5,8))
  }
}

3 scala的标识符

  • 重点是_
  • scala的标识符包括字母、数字、下划线（下划线在最前面或最后面一般都有特殊含义，自己写时需要放置最中间）。
  • scala建议使用驼峰命名法：第一个单词全部小写，其后的单词首字母大写；
  • 常量在java中一般全是全大写，scala是第一个字母大写。
  • 下划线不建议用在首、尾位置。_在scala中的含义比较多，最多的用法时通配符（用在匿名函数）和范围标识。
    • 匿名函数使用_，是在调用时使用，可以简化函数写法。
